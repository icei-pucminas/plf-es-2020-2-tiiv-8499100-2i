# 2i Inteligência Industrial

O grupo elaborou uma solução de software composta por um aplicativo móvel e um website. O aplicativo será o meio de divulgação de conteúdos da empresa e contará com fóruns de discussão sobre GD&T. Além disso, o aplicativo irá possuir uma calculadora para realizar cálculos necessários na área de GD&T. Todo o conteúdo exibido na aplicação será gerenciado na plataforma web, à qual o administrador do sistema terá acesso. Essa solução irá contribuir na divulgação da empresa para o mercado e na captação de possíveis leads, que irão se cadastrar na plataforma.

### Requisitos para a configuração do projeto:

* Front-end:
  * Node 12
  * NPM 6
* Back-end:
  * Python 3.8

### Como configurar:

1. Instalar os softwares requisitos
2. Instalar as dependências do Python e do Node
3. Iniciar o back-end pelo arquivo `__init__.py`
4. Iniciar o front-end rodando `npm run start` na raiz do front-end
